## Creating an Account

### Step 1: Navigate to Sign Up Page

1. Open the application in your web browser
2. Click on the **"Sign Up"** button or navigate to the signup page

### Step 2: Fill in Your Details

## Uploading Sales Data

### Supported File Formats

✅ **PDF Files** - Invoices, receipts, sales reports  
✅ **Excel Files** (.xlsx, .xls) - Spreadsheets with sales data  
✅ **CSV Files** - Comma-separated values files

### Step-by-Step Upload Process

#### 1. Prepare Your Files

Make sure your files contain:
- **Date**: Invoice or transaction date
- **Product/Item Name**: What was sold
- **Quantity**: How many units
- **Price**: Unit price or total amount
- **Customer Name** (optional but recommended)

#### 2. Upload Files

1. Go to the main dashboard
2. Look for the **"Upload Sales PDFs"** section in the sidebar
3. Click **"Browse files"** or drag and drop your files
4. Select one or multiple files
5. Wait for the upload to complete

#### 3. Processing

The system will automatically:
- Extract data from your files
- Validate the information
- Store it securely in your account
- Update your dashboard with new insights

### Tips for Best Results

💡 **For PDF Files:**
- Use clear, high-quality scans
- Ensure text is readable (not blurry)
- Standard invoice formats work best

💡 **For Excel/CSV Files:**
- Use column headers: Date, Product, Quantity, Price
- Ensure dates are in a consistent format
- Remove any empty rows

## Understanding the Dashboard

### Key Performance Indicators (KPIs)

At the top of your dashboard, you'll see four main metrics:

1. **Revenue** 💰
   - Total sales revenue for the selected period
   - Displayed in your currency (₹)

2. **Quantity Sold** 📦
   - Total number of items/units sold
   - Helps track inventory movement

3. **Products** 🏷️
   - Number of unique products sold
   - Shows product diversity

4. **Customers** 👥
   - Number of unique customers
   - Indicates customer base size

### Filters

Use filters to analyze specific data:

**Date Range Filter:**
- Select start and end dates
- View data for specific periods
- Compare different time frames

**Category Filter:**
- Filter by product category
- Focus on specific product lines

**Customer Filter:**
- Analyze individual customer purchases
- Identify top customers

## Troubleshooting

### Common Issues and Solutions

#### ❌ "File upload failed"

**Solutions:**
- Check your internet connection
- Ensure file size is under 50MB
- Verify file format (PDF, Excel, or CSV)
- Try uploading one file at a time

#### ❌ "No data extracted from PDF"

**Solutions:**
- Ensure PDF contains readable text (not just images)
- Check if invoice format is standard
- Try converting to Excel/CSV format
- Contact support with a sample file

#### ❌ "Login failed"

**Solutions:**
- Verify email and password are correct
- Check if email is verified (check inbox)
- Try password reset if forgotten
- Clear browser cache and cookies

#### ❌ "Dashboard shows no data"

**Solutions:**
- Upload sales files first
- Check if filters are too restrictive
- Verify files were processed successfully
- Refresh the page

#### ❌ "Charts not displaying"

**Solutions:**
- Refresh the browser page
- Clear browser cache
- Try a different browser
- Check internet connection

## FAQs

### General Questions

**Q: Is my data secure?**  
A: Yes! All data is encrypted and stored securely. Only you can access your data through your password-protected account.

**Q: Can I delete my data?**  
A: Yes, you can delete individual files or your entire account through the profile settings.

**Q: How many files can I upload?**  
A: There's no limit on the number of files, but each file should be under 50MB.

**Q: Can multiple users access the same account?**  
A: Each account is individual. For team access, each member should create their own account.

### Data Questions

**Q: What date formats are supported?**  
A: Common formats like DD-MM-YYYY, MM/DD/YYYY, and YYYY-MM-DD are automatically detected.

**Q: Can I edit data after uploading?**  
A: Currently, you cannot edit uploaded data. You can delete and re-upload corrected files.

**Q: How far back can I analyze data?**  
A: You can analyze data from any time period, as long as you've uploaded the files.

**Q: What if my invoice format is different?**  
A: The system supports multiple invoice formats. If extraction fails, try converting to Excel/CSV.

### Technical Questions

**Q: Which browsers are supported?**  
A: Chrome, Firefox, Safari, and Edge (latest versions recommended).

**Q: Can I use this on mobile?**  
A: Yes, the application is mobile-responsive, though desktop is recommended for best experience.

**Q: Do I need to install anything?**  
A: No, it's a web application. Just open it in your browser.

**Q: Is there an offline mode?**  
A: No, an internet connection is required to use the application.
